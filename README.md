# <a name="microsoft-graph-training-module---build-aspnet-core-apps-with-microsoft-graph"></a>Module de formation Microsoft Graph : créer des applications principales ASP.NET avec Microsoft Graph

Ce module vous permettra de vous familiariser avec le kit de développement logiciel (SDK) Microsoft Graph .NET dans la création d’une application Web MVC ASP.NET pour accéder aux données dans Office 365.

## <a name="lab---build-aspnet-core-apps-with-microsoft-graph"></a>Lab-créer des applications ASP.NET principales avec Microsoft Graph

Dans cet atelier, vous allez créer une application MVC ASP.NET, configurée avec Azure Active Directory (Azure AD) pour l’authentification & autorisation, qui accède aux données dans Office 365 à l’aide du kit de développement logiciel (SDK) Microsoft Graph .NET.

- [Didacticiel Microsoft Graph principal ASP.NET](https://docs.microsoft.com/graph/tutorials/aspnet-core)
